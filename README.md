# 🪙 Crypto Dashboard

A modern, clean, and real-time cryptocurrency price tracking dashboard built using **React** and **Tailwind CSS**.

---

## 🚀 Live Demo
🔗 **[View Live Web App](https://[Your-GitHub-Username].github.io/crypto-dashboard)**

---

## ✨ Features
* **Real-Time Market Data:** Displays current prices, symbols, and 24-hour percentage changes for popular cryptocurrencies (Bitcoin, Ethereum, Tether, etc.).
* **Responsive Design:** Mobile-first UI optimized for seamless viewing across mobile phones, tablets, and desktop devices.
* **Modern Dark UI:** Styled with Tailwind CSS for a crisp and premium financial dashboard aesthetic.
* **Lightweight Architecture:** Built entirely inside `index.html` using React & Babel via CDN, requiring no complex build tools or `node_modules`.

---

## 🛠️ Tech Stack
* **Frontend:** React.js (CDN)
* **Styling:** Tailwind CSS (CDN)
* **Compiler:** Babel (In-browser Transformer)
* **Hosting:** GitHub Pages
